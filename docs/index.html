Multi-Platform Demo

Helm Chart Repository

A demonstration application showcasing the same Hello World website implemented in 5 different programming languages with a collaborative canvas feature.

C# .NET 9.0 Node.js 22 Python 3.13 Java 23 Go 1.23
View on GitHub Releases

Quick Start

1. Add the Helm repository:

helm repo add multi-platform https://xiliath.github.io/multi-platform-demo
helm repo update

2. Install the chart:

helm install my-demo multi-platform/multi-platform-demo

3. For desktop Kubernetes (Docker Desktop, Minikube, kind):

helm install my-demo multi-platform/multi-platform-demo \
  --set nginx.service.type=NodePort

Features

  • Multi-platform Hello World in 5 languages
  • Real-time collaborative drawing canvas
  • WebSocket-based live synchronization
  • Unified UI theme across all platforms
  • QR code support for mobile access
  • Platform-specific routing
  • Production-ready with resource limits
  • Desktop Kubernetes optimized

Architecture

The chart deploys 7 microservices:

  • 5 platform services (ports 5000-5004)
  • WebSocket server for real-time collaboration
  • Nginx reverse proxy and load balancer

Configuration

View all configuration options:

helm show values multi-platform/multi-platform-demo
